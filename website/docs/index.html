Key Concepts

Micro-Kernel Architecture

ReactScope uses a micro-kernel architecture where the core kernel provides minimal functionality, and all features are implemented as plugins. This allows for maximum flexibility and tree-shaking.

Event-Driven Design

All component events (renders, mounts, unmounts, props changes) are emitted through a central event bus. Plugins subscribe to these events to perform their specific functions.

Centralized Metrics Store

All metrics are stored in a central store, making it easy to query, export, and visualize performance data across your entire application.

Shadow DOM Dashboard

The visual dashboard renders inside a Shadow DOM to ensure complete CSS isolation from your application. This prevents any style conflicts.
